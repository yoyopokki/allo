# Мессенджер - Документация

## Структура

### Компоненты

- **MessengerHeader** - шапка приложения с кнопками поиска и настроек
- **MessengerUserList** - левая колонка со списком пользователей и поиском
- **MessengerUserListItem** - элемент списка пользователей с аватаром, именем, статусом онлайн, последним сообщением и кнопкой звонка
- **MessengerChatWindow** - правая колонка с окном чата
- **MessengerMessageItem** - элемент сообщения в чате
- **MessengerMessageInput** - поле ввода сообщения с кнопкой отправки

### Store (Pinia)

`useMessengerStore` управляет состоянием:
- Список пользователей
- Выбранный пользователь
- Чаты с сообщениями
- Отправка сообщений

### Типы

```typescript
interface User {
  id: string;
  firstName: string;
  lastName: string;
  avatar?: string;
  isOnline: boolean;
  lastMessage?: string;
  lastMessageTime?: Date;
}

interface Message {
  id: string;
  senderId: string;
  receiverId: string;
  content: string;
  timestamp: Date;
  isRead: boolean;
}
```

## Использование

Перейдите на `/messenger` для просмотра интерфейса мессенджера.

### Функционал

- ✅ Список пользователей с аватарами и статусами
- ✅ Отображение последнего сообщения и времени
- ✅ Индикатор онлайн-статуса (зеленая точка)
- ✅ Кнопка звонка у каждого пользователя
- ✅ Окно чата с сообщениями
- ✅ Поле ввода с кнопкой отправки
- ✅ Отправка по Enter
- ✅ Автопрокрутка к новым сообщениям
- ✅ Адаптивный дизайн с Tailwind CSS
- ✅ Темная тема (через Nuxt UI)

## Моковые данные

При первом открытии загружаются 3 тестовых пользователя с демо-сообщениями.
